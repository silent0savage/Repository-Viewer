Для того, чтобы авторизоваться требуется ввести Personal Access Token вашего профиля Github.

Чтобы сгенерировать токен перейдите в Developer Settings -> Personal Access Tokens -> Fine-grained tokens -> Generate Token

Авторизация будет сохранена при закрытии приложения.

Пользовать может просматривать базовую информацию о своих репозиториях, такие как: название, описание, просмотры, форки, readme.

Библиотеки:

remote - Ktor Client,

di - Koin

token storage - Shared Preferences.
